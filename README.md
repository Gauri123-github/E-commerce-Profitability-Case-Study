E-Commerce Sales Analytics using Python, MySQL & Power BI

Project Overview-
This project analyzes e-commerce sales data to uncover business insights related to revenue, customer behavior, product performance, payment methods, and regional sales. The project demonstrates an end-to-end data analytics workflow using Python, MySQL, and Power BI

Problem Statement-
E-commerce businesses generate large volumes of transaction data. The objective of this project is to clean, analyze, and visualize this data to help businesses understand sales performance, customer purchasing patterns, and operational efficiency

Objectives-
Analyze sales performance
Identify top-selling products
Compare revenue across categories and regions
Analyze customer demographics
Evaluate payment methods and return rates
Build an interactive dashboard for business insights

Key Business Insights-
Electronics generated the highest revenue.
UPI was the most frequently used payment method.
South region contributed the highest sales.
Return rates were higher for certain categories.
Customers aged 25–35 placed the highest number of orders.
